Support for Illumina Genome Analyzer

(Nov 2008) Samples and derived data from ultra-high throughput sequencing (e.g. ChIP-Sequencing) can now be annotated in MIMAS.

Support for Illumina BeadArrays

(Jun 2008) The MIMAS array library has been extended to support Illumina BeadArrays as well as the latest Affymetrix GeneChip types.

MIMAS moves to Vital-IT

(Dec 2006) MIMAS has been transferred to the Vital-IT informatics platform.

Sharing the annotation process

(Oct 2006) MIMAS users can now control who is allowed to view and/or modify the annotation of their samples. This allows a smoother annotation process where the different actors (sample provider, platform technician, data analyst) can supply different parts of the complete annotation.

Direct submissions to ArrayExpress

(Sep 2006) MIMAS is designed to be MIAME-compliant, so that data can be seamlessly exported to the EBI ArrayExpress data repository. Curated experiments stored in MIMAS can now be exported with a single click to an archive containing measurement files and annotation in an Excel spreadsheet. This spreadsheet can be reviewed and the archive can be directly sent to the ArrayExpress curators. Within a few days the ArrayExpress team provides an accession number and a password protected account for reviewers required for publication of array data by most journals.

MIMAS data displayed in GermOnline

(Aug 2006) GermOnline is a cross-species knowledgebase that provides GeneChip and tiling microarray data relevant for the cell cycle and gametogenesis. The report page integrates data on DNA sequence, RNA concentration, length and exon composition as well as information on protein function, structure and network interaction. 26 expression datasets stored in MIMAS are displayed in the report pages (example: Yeast SPO11 page).

Yeast tiling arrays

(Jul 2006) MIMAS now supports data obtained from yeast tiling arrays. Two designs are supported, the Affymetrix S. cerevisiae Tiling 1.0R Array and the Affymetrix Scerevisiae_tlg custom chip. Tiling arrays cover the entire genome with overlapping probes and allow the identification of novel transcripts, the determination of transcript length, and chromatin immunoprecipitation (ChIP) assays.

MIMAS paper published

(May 2006) The article: Hermida et al., MIMAS: an innovative tool for network-based high density oligonucleotide microarray data management and annotation has been published in BMC Bioinformatics 7:190 (2006).
